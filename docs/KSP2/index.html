Kerbal Space Program 2 API Documentation

This is a project to automatically generate documentation for Kerbal Space Program 2.

The documentation is based on the game's assemblies, however they are not shared as part of the process in any way.

Notes

I had to write a script to add the namespace DefaultNamespace to every class. This is not how it is in-game or in the assemblies. Just be wary of that.

Important

This documentation is NOT affiliated in ANY way with Private Division, Kerbal Space Program 2, or any of its developers. In addition, this may not always be completely up-to-date with the latest version of the game, so use at your own risk.
